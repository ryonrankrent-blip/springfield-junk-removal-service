## What actually appears to help competitors

The useful portion of the leading profile is not the raw link count. The defensible patterns are:

1. A Springfield real-estate professional's contractor/partner resource page.
2. Local business or community directories with real Springfield coverage.
3. Real-estate/estate/property-management association or affiliate directories.
4. Earned local editorial/recommendation pages.
5. Useful local disposal, cleanout, landlord, moving, or estate resources that can attract contextual links.

The profile also contains many patterns that should not be copied: paid press-release syndication, PBNs, mass directories, reciprocal badge schemes, irrelevant blog/forum pages, scraper indexes, foreign or unrelated sites, and exact-match link campaigns.

## Recommended first approval batch

No link acquisition should begin yet. The smallest defensible next batch is:

1. Build a read-only list of 10–15 real Springfield-area realtor, property-management, moving, estate, storage, and landlord resource pages.
2. Score each page for local relevance, editorial ownership, indexation, outbound-link quality, and fit with the new disposal guide.
3. Prepare two outreach drafts: one for the disposal guide and one for the landlord/turnover cluster.
4. Stop for approval before sending any message.

Do **not** begin with general directory volume. The current site already attracts unsolicited spam, and the competitive advantage is more likely to come from a few legitimate local/editorial links than from copying hundreds of low-quality referring domains.
