## Evaluation rule

An outreach target must have all three:

1. A specific existing page or clearly verifiable content gap.
2. A Springfield resource that improves the target’s user experience.
3. An honest pitch that does not imply an operating crew, provider relationship, endorsement, or service history.

General “please link to us” outreach is rejected.
