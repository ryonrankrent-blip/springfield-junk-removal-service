## 6. Suitability assessment

### Recommendation: **Suitable only after specific missing facts are resolved**

ShowMeLocal is potentially useful for a local citation because it supports free listings, relevant waste/junk-related categories, phone/website/profile fields, and a dashboard privacy control to hide street address. However, unlike Brownbook, ShowMeLocal's official guidance requires a real physical business address and rejects virtual-only or invalid address submissions.

Public address visibility is addressable through ShowMeLocal's dashboard privacy controls, but the underlying legitimate physical address requirement remains unresolved for Springfield. The current model also lacks verified local operator/service-provider facts and a renter/operator handoff plan.
